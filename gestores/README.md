## Descripción

Desarrollo de dos sistemas de gestión utilizando Python con POO, módulos y excepciones personalizadas.

## Estructura del proyecto

```
tp-programacion2/
├── ejercicio1_biblioteca/   # Sistema de gestión de biblioteca
│   ├── modelos.py
│   ├── biblioteca.py
│   └── main.py
├── ejercicio2_facultad/     # Sistema de gestión de facultad
│   ├── modelos_facultad.py
│   ├── facultad.py
│   └── main_facultad.py
├── docs/                    # Documentación y entrega
│   └── entrega_prog2.docx
├── main_general.py          # Menú principal que integra ambos sistemas
└── README.md
```

## Cómo ejecutar

Para correr ambos sistemas desde el menú principal:

```
python main_general.py
```

O ejecutar cada ejercicio por separado:

```
python ejercicio1_biblioteca/main.py
python ejercicio2_facultad/main_facultad.py
```
